This directory contains input data files for this analysis/paper

These include:

- all_reptarenavirus_L_seqs_2025_10_31.gb: all reptarenavirus L segment sequences from the NCBI nucleotide database on Oct 31, 2025
- all_reptarenavirus_S_seqs_2025_10_31.gb: all reptarenavirus S segment sequences from the NCBI nucleotide database on Oct 31, 2025
- coding_complete_L_for_trees.gb: the coding-complete L segment sequences included in trees
- coding_complete_S_for_trees.gb: the coding-complete S segment sequences included in trees
- GP_CDS.fasta: The GPC coding sequences (CDS) from coding_complete_L_for_trees.gb
- L_CDS.fasta: The L protein coding sequences (CDS) from coding_complete_L_for_trees.gb
- NP_CDS.fasta: The NP coding sequences (CDS) from coding_complete_S_for_trees.gb
- Z_CDS.fasta: The Z protein coding sequences (CDS) from coding_complete_S_for_trees.gb
- VMR_MSL40.v2.20251013.xlsx: The ICTV master species list file, MSL40.v2, [downloaded from the ICTV website](https://ictv.global/msl)
